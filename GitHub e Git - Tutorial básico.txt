Comando básico do git para subir arquivos:

- cd desktop
- cd nomeDaPasta
- git init
- git commit -m 'Digite seu comentário mensagem'
- git remote add origin linkDoRepositorioNoGit
- git push -u origin master

Para atualizar um arquivo e upá-lo no git:
- Altere o arquivo desejado e salve-o
- git status
- git add *
- git status (se estiver verde)
- git commit -m 'Comentário sobre a alteração'
- git push
- git log (Para ver os commits)
